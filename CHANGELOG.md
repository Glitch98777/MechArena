# Changelog

All notable updates to Mech Arena. Newest first.

## v1.0.0 — Initial public release
- **18 mechs** (5 core, rare tier, 3 Legendary, 3 Mythic) with unique silhouettes, abilities, and glowing trim on top-tier chassis.
- **17 weapons** including 7 legendary/mythic overpowered options.
- **10 buyable/equippable abilities**; per-mech loadouts (weapon + ability) that persist.
- **3-mech squad** with mid-battle swap hotbar; downed mech auto-deploys the next.
- **Rank system** (Recruit → Legend) with home-screen badge, XP from battles and unlocks, rank-up celebration.
- **Rank + credit gating** on legendary/mythic gear, tuned to be a reachable grind (−15% price / −2 rank pass applied).
- **Progression-scaled difficulty** — enemy mech quality, weapons, health, and count scale to your rank + hangar.
- **Line-of-sight cover** so obstacles block shots; **drag-to-look** camera with protected button zones.
- **Daily reward crate** with open animation; **win/lose reward tiers**.
- **Backup / Restore** save data to a JSON file from the main menu.
- **Procedural SFX + menu ambience** (weapon-specific fire, hits, explosions, abilities, UI, stingers, rank-up jingle).
- **Bloom / vignette** post-processing, camera shake, kill feed, pause menu, rematch.
- **Tesla resist** — APEX survives one insta-kill (drops to 10%), Colossus/Omega survive one (drop to 25%).
- Release-signed **IL2CPP / arm64** APK for broad device compatibility.
